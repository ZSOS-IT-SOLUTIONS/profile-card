# Nagadeepak V - Profile Card

A modern, responsive personal profile card featuring a glassmorphism design, background video, and neon light animations.

## Technologies Used
- HTML5
- CSS3 (Animations, Flexbox, Glassmorphism)
- JavaScript

## Features
- **Glassmorphism UI**: Semi-transparent card with blur effect.
- **Background Video**: Engaging video background (`video.mp4`).
- **Neon Animations**: Moving neon lines in the background for a dynamic look.
- **Responsive Design**: Adapts to different screen sizes.

## How to View
Simply open the `index.html` file in any modern web browser.
